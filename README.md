# Chess Master

This is a full-featured chess application that allows you to play against other players or the computer. It also includes a unique feature that allows you to use the Gemini API to analyze your games.

## Core Functionalities

This chess application provides a comprehensive set of features for chess enthusiasts:

-   **Player vs. Player Mode:** Challenge your friends to a game of chess in a real-time, online setting. The application ensures a smooth and seamless multiplayer experience.

-   **Player vs. Computer Mode:** Hone your skills by playing against a computer opponent. This mode is perfect for practicing your strategies and improving your gameplay.

-   **AI-Powered Game Analysis:** Leverage the power of Google's Gemini API to analyze your completed games. This feature provides you with valuable insights into your gameplay, helping you to identify your strengths and weaknesses.

## Getting Started

**Live demo:** https://chess-865.pages.dev/

To use the AI analysis feature, you will need to add your own Gemini API key.

1.  **Log in:** Access the application and log in to your account.
2.  **Navigate to Settings:** After logging in, click on the settings icon.
3.  **Add API Key:** In the settings menu, you will find an option to add your Gemini API key.

## Technologies Used

This project is built with the following technologies:

### Frontend

-   **React:** A JavaScript library for building user interfaces.
-   **socket.io-client:** Enables real-time, bidirectional and event-based communication.
-   **lucide-react:** A library of simply designed, beautiful and consistent icons.

### Backend

-   **Node.js:** A JavaScript runtime built on Chrome's V8 JavaScript engine.
-   **Express:** A fast, unopinionated, minimalist web framework for Node.js.
-   **Socket.IO:** Enables real-time, bidirectional and event-based communication.
-   **PostgreSQL:** A powerful, open source object-relational database system.
-   **bcryptjs:** A library for hashing passwords.
-   **jsonwebtoken:** An implementation of JSON Web Tokens.
-   **cors:** A package for providing a Connect/Express middleware that can be used to enable CORS with various options.
-   **dotenv:** A zero-dependency module that loads environment variables from a `.env` file into `process.env`.
-   **uuid:** A package for creating RFC4122 UUIDs.
-   **compression:** A Node.js compression middleware.
-   **helmet:** A package that helps secure Express apps by setting various HTTP headers.
